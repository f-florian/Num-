Detailed Description

Information on weights for interpolatory opertions on the Lagrange basis on a fixed set of points.

Stores a mesh of [0,1] and weights wich can be used to compute ∫_0^1 f(x) d x and f'(y) for y a point of the mesh,.and g(x) for g a polyneomial of the basis and x ∈ [0,1] The weights are stored for the barycentric representation, which reduces the error w.r.t. standard Lagrange representitatio
