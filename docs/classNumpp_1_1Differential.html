Detailed Description

Information on weights for interpolatory opertions on the Lagrange basis on a fixed set of points.

Stores the Mesh, and weights wich can be used to compute ∫_0^1 f(x) d x and f'(y) for y a point of the mesh, and g(x) for g a polynomial of the basis and x ∈ [0,1] The weghts are stored for the barycentric representation, which reduces the error w.r.t. standard Lagrange representitatio
